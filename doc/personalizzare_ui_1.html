Personalizzare la UI - 1
Sicuramente, anche se non siete dei fini esteti, vorrete ridefinire gli stili CSS rispetto a quelli forniti di default; questi, di fatto, non sono definiti: esiste solo un template pressoché vuoto che dovrete riempire se non volete un grazioso risultato di tipo Deutsche Demokratische Republik. Come fare l'abbiamo già visto: create il foglio CSS dell'applicazione dentro la vostra UI e modificatelo a piacere 
$ cd /var/www/testwalibs
$ mkdir ui
$ mkdir ui/css
$ pico ui/css/testwalibs.css
			

/****************** testwalibs.css ***************************************/

	margin: auto;
	margin-top: 50px;
	width: 80%;
	border: #000000 solid 1px;

	}
