										mixed caricaDaSql(
																																														string
										$sql																					, 											[int
										$nrRighe = null]																					, 											[int
										$rigaIniziale = 0]																	)
						

			



	
-
Esegue una query SQL, costruisce le strutture dati relative alle colonne  e alle righe e restituisce un array di righe waRecord.
	

		

							

					
																																			ritorna:  (boolean | array) : se esito positivo ritorna l'array delle righe restituite  dalla query; se esito negativo torna false; per conoscere l'esatto esito  della query occorre invocare il metodo nrErrore														
				

					

	


		
    
Parametri:

    

		

							

					string  
					$sql  
					SQL da eseguire
				

							

					int  
					$nrRighe  
					numero max di righe che la query deve restituire
				

							

					int  
					$rigaIniziale  
					numero di righe di cui effettuare lo skip (offset)

-
Ritorna il nome del campo che e' chiave primaria del record  attenzione: potrebbero darsi casi in cui esistono piu' chiavi  primarie (viste) o in cui non sia stata selezionata nessuna chiave  primaria; questo metodo ritorna il nome del primo campo che e'  chiave primaria, se esistente. Sta al programmatore calare correttamente  la chiamata a questo metodo nel contesto in cui viene chiamato
