waApplicazione 3.0

Il package waApplicazione fornisce un contenitore di base da cui partire per realizzare applicazioni web-based e database-independent.

Non fornisce la struttura, lo scheletro di una applicazione, quanto piuttosto un criterio per realizzare applicazioni con una metodologia ordinata e per generare codice quanto più possibile leggibile, documentabile e riutilizzabile.

Il package, ossia la sua unica classe waApplicazione, non è di fatto utilizzabile così come è fornito: dovrà essere sempre esteso tramite una classe derivata che conterrà i metodi e le proprietà comuni alle diverse pagine, e a propria volta l'estensione dovrà essere estesa tramite le sue eventuali sezioni (es.: valutazione, progettazione, ecc.) e infine tramite le pagine, che essendo quindi istanze dell'applicazione erediteranno tutti i metodi e le proprietà di tutte le classi parent.

Un esempio di utilizzo della classe prevede quindi la creazione di almeno due moduli: il primo definisce la classe della vostra applicazione (nel nostro esempio: testwalibs), che conterrà i metodi/proprietà comuni a diverse pagine della vostra applicazione e ridefinirà tramite overloading il comportamento dei metodi di waApplicazione laddove necessario, e il secondo (nel nostro esempio pagina1.php) che conterrà i metodi/proprietà della singola pagina: 

<?php

//****************** testwalibs.inc.php ***************************************

include dirname(__FILE__) . "/walibs3/waapplicazione/waapplicazione.inc.php";

class testwalibs extends waApplicazione
	{
	// dati dell'utente loggato: proprietà della mia applicazione
	var $datiUtente;

	function __construct()
		{
		// valorizzazione delle proprietà già definite da waApplicazione
		$this->nome = "testwalibs";
		$this->titolo = "Voglio testare le waLibs";

		$this->inizializza();

		// valorizzazione delle proprietà della mia applicazione
		$this->datiUtente = & $_SESSION['dati_utente'];
		}
 
	}
?>
				

<?php

//****************** pagina1.php ***************************************

include dirname(__FILE__) . "/testwalibs.inc.php";

class pagina extends testwalibs
	{
	function faiQualcosa()
		{			
		$this->aggiungiElemento("ciao mondo", "titolo");
		$this->mostra();
		}
 
	}

$pagina = new pagina();
$pagina->faiQualcosa();
?>

Naturalmente potete personalizzare molto meglio la vostra applicazione e renderla molto più complessa e adatta alle vostre esigenze. Per scoprire come vi invitiamo a scorrere le descrizioni dei diversi proprietà/metodi che la classe vi mette a disposizione.
