									$valore =  ''										
								
							
						

					

			



			
			
valore del controllo
il valore assume significati diversi a seconda del controllo. Alcuni esempi: 
  • waTesto, waAreaTesto, waValuta, waData,
   waDataOra, waCaricaFile -> e' il valore di default, che puo' essere
   ricoperto da cio'che viene trovato sul db
  • waEtichetta, waBottone, waCornice, waPortale ->
   e' la caption
  • waSelezione/waOpzione -> viene selezionato l'elemento corrispondente al valore passato,
   purche' l'array degli elementi selezionabili (ancorche' proveniente da una
   query) lo contenga
  • waLogico -> 1 se checked - 0 o vuoto altrimenti
  • waMultiSelezione, lista -> e' un array contenente i valori degli elementi selezionati
			

				

					

						Tipo:  
						mixed
