waCaricaFile

classe per la gestione di un controllo complesso che permette di:

  • caricare un file (upload)
  • scaricare/vedere inline il file caricato
  • sostituire un file
  • eliminare un file precedentemente caricato



[ Top ]

Proprietà

$dimensioniFile =    0

size del file in Kb

Se valorizzato, viene mostrato il size del file accanto al nome del file precedentemente caricato.

Tipo:   integer

[ Top ]

$mostraNomeFile =     true

indica se visualizzare il nome del file precedentemente caricato

Del file viene ovviamente mostrato solamente il basename.

Tipo:   boolean

[ Top ]

$paginaVisualizzazione =    ""

URL per visualizzazione file

e' l'URL di una pagina tramite cui e' possibile prendere visione del file Se valorizzato, dovra' anche contenere tutti i parametri per accedere al file corretto (la classe non aggiunge nulla).

Tipo:   string

[ Top ]

Metodi

boolean daEliminare( )

Restituisce true se l'utente ha richiesto la eliminazione del file esistente

Si usa in fase di ricezione dei dati, non durante il display del modulo.

[ Top ]
string dammiValoreTmp( )

Restituisce il path del file temporaneo caricato dall'utente

Si usa in fase di ricezione dei dati, non durante il display del modulo.

[ Top ]
boolean daSalvare( )

Restituisce true se c'e' effettivamente qualcosa da salvare

Si usa in fase di ricezione dei dati, non durante il display del modulo.

[ Top ]
mixed erroreCaricamento( )

ritorna il codice di errore avvenuto durante il caricamento o false in caso di nessun errore

Si usa in fase di ricezione dei dati, non durante il display del modulo.

ritorna:  false in caso di nessun errore; altrimenti il codice errore generato da PHP durante il caricamento
[ Top ]
boolean salvaFile( string $destinazione )

salva il file nella destinazione richiesta

Si usa in fase di ricezione dei dati, non durante il display del modulo.

ritorna:  true = ok

Parametri:

string   $destinazione   puo' essere sia un nome completo di file, sia il nome di una directory; in questo secondo caso, il nome del file sara' quello del file temporaneo attribuito dall'engine PHP
