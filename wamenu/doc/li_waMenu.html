waMenu 3.0

 

Classe per la gestione di un menu

Features:

  • completa governabilita' del menu mediante foglio di stile XSLT
  • gia' compresi nel package i fogli XSLT (UIS) per:
    • formato ad albero
    • orizzontale e verticale
    • usabile (senza uso di javascript)
    • posizionamento statico o dinamico
    • always on top
    • a scomparsa (automatica e manuale).

Come si usa il package? Seguite questo esempio di codice PHP: 

<?php

// includete il package waMenu
include "wamenu.inc.php";

// create l'oggetto e apritelo
$m = new waMenu();
$m->apri();

// aprite le diverse sezioni, anche nestate, e riempitele
// con le voci di vostro gradimento
$m->apriSezione("PHP");
	$m->aggiungiVoce("Manuale",  "http://it.php.net/manual/it/");
	$m->aggiungiVoce("PHPClasses", "http://www.phpclasses.org/");
	$m->apriSezione("Varie");
		$m->aggiungiVoce("weberdev",  "http://www.weberdev.com/");
		$m->aggiungiVoce("freescripts", "http://www.freescripts.com/");
	$m->chiudiSezione();
$m->chiudiSezione();
$m->apriSezione("Javascript");
	$m->aggiungiVoce("reference", "http://javascript-reference.info/");
$m->chiudiSezione();
$m->apriSezione("Logout", "logout.php");
// chiudete le diverse sezioni che avete aperto e chiudete il menu
$m->chiudiSezione();
$m->chiudi();
$m->mostra();
?>
