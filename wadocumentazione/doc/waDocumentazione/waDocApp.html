Dettaglio

applicazionetest

questa classe conterra' tutte le proprieta' e i metodi comuni della nostra specifica applicazione. Deriva da waApplicazione, dalla quale quindi eredita l'interfaccia programmatica (proprieta' e metodi); a questa noi potremo aggiungere i nostri metodi applicativi e se necessario modificare il comportamento della classe di default mediante l'override dei metodi.

array dammiLista( string $table , string $idFieldName , string $descriptionFieldName )

restituisce una lista da associare a un controllo di classe waSelezione

parendo da una tabella di database, restituisce un array associativo in cui la chiave e' l'dentificativo del record e il valore la descrizione del record.

Tipicamente questo array viene associato ad un controllo di classe waSelezione all'interno di un modulo nel quale occorre scegliere il valore di un campo foreign-key

Parametri:

string   $table   nome della tabella del db
string   $idFieldName   nome del campo identificativo univoco del record
string   $descriptionFieldName   nome del campo che contiene una descrizione umanamente conprensibile del record

waModulo dammiModulo( )

restituisce un modulo standard dell'applicazione

questo metodo e' molto comodo in una applicazione, perche' permette di:

  • definire in un unico punto gli attributi di TUTTI i moduli dell'applicazione
  • qualora in fase di lavorazione si decidesse di non utilizzare direttamente la classe waModulo, bensi' una sua derivata, e' sufficiente intervenire in questo unico punto dell'applicazione, anziche' in tutte le pagine che contengono un modulo
